# ToDoApp-React-Redux

### Installation

```
1. Clone the project in a directory using command -

        git clone https://github.com/akar444/ToDoApp-React-Redux.git

2. After cloning the project navigate to directory ToDoApp-React-Redux
3. Run command
        npm install
   to install all node dependencies.

4. Run command
        webpack

5. Now run command
        npm start
   to start the server.It will start the server at
        http://localhost:4000

```
